我的2020年理财与职业规划

2020-03-25 | 规划

资产

月固定支出：

  1. 房租+水电 2200
  2. 日常饮食 2500+
  3. 车贷 4200 (还剩余14个月)
  4. 停车费+油费：1000
  5. 车保险月均：600
  6. 抽烟： 600

固定收入：

  1. 工资 25000
  2. 奖金（约3w30.8= 7W）

月可支配开支：
1.4W
年可支配约

现金or可用现金资产：

  1. 比特币 10W
  2. 出借款 2.6W
  3. 公积金 8W （只有买房时提取）
  4. 医保个人账户 2.5W （意外情况支用）

目标：
最低目标：2022年中前，在东莞稍偏点地带买一套房，需准备现金30W（百万总价左右，东莞首付，装修另计）
理想目标： 150W资金，在深圳付首付

理财规划：
最低目标保障：
2年后，公积金账户13W现金。
自己需准备17W现金，约为2021年底的奖金+2020前半年的收入

最大承受风险：
2020年-2021年年的月工资收入，归0。

投资方法：
半年时间，持续至10月，每月定投2W至比特币，持续半年，账户预计会有20W资产。如若具有在深圳买理想房子条件，需要涨幅500%。

10月后，自己攒下来，备用。
