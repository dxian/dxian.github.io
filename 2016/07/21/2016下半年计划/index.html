2016下半年计划

发表于 2016-07-21   |   分类于 心情   |     |   阅读次数

世界上，从来，从来没有什么非做不可的事情。
你的人生属于你自己，不是为了说服别人，向别人证明、夸饰、博得别人的好感而存在。
无论多爱或多惧怕那个人。

2016年1月1日许下的目标

  1. 成为一名开发
  2. 攒下20万
  3. 学习英语到能简单的听说读写
  4. 告别单身

除了工作，一无所成，呵呵。。。
英语几月没翻，钱倒是在慢慢的增多，单身···

2016年8月1日目标如下，期望在过年前达到：

  1. 攒下20万
    谁找我借钱都不借了~
  2. 养成一个学习的习惯
    这30天来，我每天晚上都会读一篇圣经诗篇，同理，我为什么不可以用来学习技能呢？有了习惯，英语还不是小case?
  3. 告别单身
    一定要鼓起勇气~，俺也很优秀不是？
  4. 受洗成为基督徒
    信？不信？能得心宁平静，就足以，何必和别人争论到底有没有神呢？
  5. 会学做计划
    学会把要做的事情量化，可以量化后就可以拆分，就可以分给别人来做，为以后管理打基础。

现在，我越发感觉我的内心知识跟不上我的发展层次了，现在我是随时会走上管理岗位或承担更多的责任的，但我的内心还不够强大，情绪不够稳定，今年以多看书、多接触人为主。

上次在简书上看到一个很受用四个能力:”主动，善学，优化，总结。”
善学包含：学技术，学管理思想，学通用技能。
我很需要锻炼我的主动性，无论是职场上还是生活上，因为主动性强的人受欢迎，且能给更多人带来益处。

未来展望

1年后的我： 要英语能达到普通的听说读写
3年后的我： 成为一名经理or总监
5年后的我： 创业
